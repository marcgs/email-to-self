# email-to-self

A sample project built on Azure for scheduling emails to be sent in the future.

[www.emailtoself.ch](https://www.emailtoself.ch)

# Cloud technologies in use
* [Azure Functions](https://docs.microsoft.com/en-us/azure/azure-functions/)
* [Cosmos DB](https://docs.microsoft.com/en-us/azure/cosmos-db/)
* [Azure Active Directory](https://docs.microsoft.com/en-us/azure/active-directory/fundamentals/active-directory-whatis)
* [Azure CDN](https://docs.microsoft.com/en-us/azure/cdn/cdn-overview)
